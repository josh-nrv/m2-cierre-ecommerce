# Proyecto de finalización Módulo 2 - Sitio E-Commerce

El proyecto consiste en una página de ecommerce ficticia, con el fin de demostrar habilidades en html, css, bootstrap.

## Principales secciones de la página

- Barra de navegación.
- Galería de productos.
- Barra lateral pegajosa con filtros
- Página de detalle de producto
- Página de carrito
- Footer
